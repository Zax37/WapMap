What's new

v0.4.0 (b33)

  1. Full polish translation(thanks Pejti, SchwarzerHund).
  2. Brought back basic support of Gruntz and GetMedieval.
  3. New settings window, allowing configuring WapMap for different games.
  4. New easy editor for decoration elements.
  5. New easy editor for elevators.
  6. New easy editor for floor spikes / saw blade.
  7. Enhanced easy editor for checkpoints.
  8. Enhanced easy editor for crab nests.
  9. Allowed editing WaterRock logic with spring boards easy editor.
  10. Allowed creating empty crates in crate easy editor.
  11. Allowed closing current tool in tile mode / other open windows with escape key.
  12. Allowed opening MDI context menu at any position, not just on the tabs (so you can reopen closed tab even if you closed last one).
  13. Added hotkey for toggling invert flag(I). Allowed using M/I keys while easy editor is open.
  14. Added hotkeys for saving (CTRL + S), saving as (CTRL + SHIFT + S) and closing (CTRL + W) current document tab.
  15. Added hotkey for reopening last closed (CTRL + SHIFT + T) document tab and creating new one (CTRL + N).
  16. Added possibility to select multiple documents to open at once.
  17. Added tooltips to most of the interface elements.
  18. Added support of recently opened documents for Windows 8 and 10.
  19. Grouped enemies into one button. Added PunkRat to enemies list.
  20. Treasure easy editor will automatically update "Add glitter" option to selected treasure's default value.
  21. Easy editor window positions no longer reset when "Add next" button is used.
  22. The editor will no longer prompt question if it should recalculate rects when moving elevators.
  23. Minor visual tweaks ("New Map" menu, mode menu, easy editors, document tabs).
  24. Removed unused fields from map settings window.
  25. Fixes and improvements to tiles text edit(auto selecting next tile, F for filled tile...).
  26. Fixed bug with command menu "Z coordinate" option opening logic editor / crashing.
  27. Fixed "Show tile properties" option crashing when tile picker is open.
  28. Fixed object added with "Add next" button not getting destroyed upon cancel.
  29. Fixes and improvements to all elevators movement preview.
  30. Fixed crash happening when the corner of context menu is clicked.
  31. Fixed tile picker leaving trail of tile ids.
  32. Fixed random powder keg / bush spawning in new levels based on The Battlements / Township.
  33. Fixed crash on clipboard preview for objects cut out of map.
  34. Fixed objects disappearing from clipboard after they are deleted from map.
  35. Fixed pencil / fill tool triggering together with tile pipette.
  36. Fixed crashes after adding / removing layers.
  37. Fixed tilesets list for new layer being empty until you select any other layer.
  38. Fixed treasure randomizer crashing on containers that can contain only one item.
  39. Fixed warp / gem destination preview for selected enemy.
  40. Fixed mirrored treasure flickering when preview / treaser picker is visible.
  41. Fixed some menu options staying enabled after switching from open document to home screen, causing crash when clicked.
  42. Fixed path elevators crashing when inverting unsaved path. Disabled x/y inverting unclosed paths, which doesn't work correctly right now.
  43. Fixed application / context menu not closing when it should be (and closing when it shouldn't).
  44. Fixed scripts creation not working when LOGICS folder does not exist.
  45. Fixed tile mode tools placing tiles even when you click outside of WapMap's window.
  46. Fixed fill tool crashing when used on big areas, like new blank document.
  47. Fixed values shown in map settings window not resetting after cancel.
  48. Fixed map being marked as changed even when placing new object was cancelled.
  49. Fixed map not being marked as changed when object is moved with keyboard input.
  50. Fixed multiple problems with recently used / closed documents (some of which were crashfixes).

v0.3.3 (b32)

  1. Fixed "WapMap is already running." message showing up instead of opening documents, when multiple opened simultaneously.
  2. Fixed opening documents through command-line(/open file with).
  3. Fixed Claw Level 12-based maps not loading properly because of unsupported PCX tile files.
  4. Temporarily disabled caching of small layers, until it's fixed.
  5. Fixed custom logics not being loaded into WapMap.
  6. Removed internal code editor for custom logics. Open default editor for lua extension instead.
  7. Allowed opening documents with drag & drop.
  8. Temporarily disabled mapshot and some other menu options, which needs to be reimplemented.
  9. Fixed MDI tab context menu (reload, close all others) actions being performed on active document instead of the one actually selected.
  10. Disabled reload option for newly created documents, until they are actually saved.
  11. Added button to first run welcome message, suggesting to set Claw install directory.
  12. Reimplemented the automatic update functionality.
  13. Set "Add back and front planes" option in "New map" window pre-selected by default.

v0.3.2 (b31)

  1. Fixed welcome page buttons.
  2. Fixed the browse button in options, allowing to pick Claw install directory.
  3. Fixed crash when opening brush tool.
  4. Fixed crash when closing text easy editor without saving.
  5. Fixed crash when some settings are not set.
  6. Fixed crash when browsing empty tiles or images data bank.
  7. Fixed some other random bugs, which could impact the performance.
