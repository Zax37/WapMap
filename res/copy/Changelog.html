What's new

v0.4.0 (b33)

  1. Full polish translation(thanks to Pejti).
  2. New easy editor for decoration elements.
  3. New easy editor for elevators.
  4. New easy editor for floor spikes.
  5. Enhanced easy editor for crab nests.
  6. Added hotkey for toggling invert flag(I). Allowed using M/I keys while easy editor is open.
  7. Added possibility to select multiple documents to open at once.
  8. Added tooltips to most of the interface elements.
  9. Grouped enemies into one button.
  10. Easy editor window positions no longer reset when "Add next" button is used.
  11. The editor will no longer prompt question if it should recalculate rects when moving elevators.
  12. Minor visual tweaks ("New Map" menu, mode menu, easy editors, document tabs).
  13. Fixed bug with command menu "Z coordinate" option opening logic editor / crashing.
  14. Fixed "Show tile properties" option crashing when tile picker is open.
  15. Fixed object added with "Add next" button not getting destroyed upon cancel.
  16. Fixes and improvements to all elevators movement preview.
  17. Fixed crash happening when the corner of context menu is clicked.
  18. Fixed tile picker leaving trail of tile ids.
  19. Fixed random powder keg / bush spawning in new levels based on The Battlements / Township.
  20. Fixed crash on clipboard preview for objects cut out of map.
  21. Fixed objects disappearing from clipboard after they are deleted from map.
  22. Fixed pencil / fill tool triggering together with tile pipette.
  23. Fixed crashes after adding / removing layers.
  24. Fixed tilesets list for new layer being empty until you select any other layer.
  25. Fixed treasure randomizer crashing on containers that can contain only one item.
  26. Fixed warp / gem destination preview for selected enemy.
  27. Fixed mirrored treasure flickering when preview / treaser picker is visible.

v0.3.3 (b32)

  1. Fixed "WapMap is already running." message showing up instead of opening documents, when multiple opened simultaneously.
  2. Fixed opening documents through command-line(/open file with).
  3. Fixed Claw Level 12-based maps not loading properly because of unsupported PCX tile files.
  4. Temporarily disabled caching of small layers, until it's fixed.
  5. Fixed custom logics not being loaded into WapMap.
  6. Removed internal code editor for custom logics. Open default editor for lua extension instead.
  7. Allowed opening documents with drag & drop.
  8. Temporarily disabled mapshot and some other menu options, which needs to be reimplemented.
  9. Fixed MDI tab context menu (reload, close all others) actions being performed on active document instead of the one actually selected.
  10. Disabled reload option for newly created documents, until they are actually saved.
  11. Added button to first run welcome message, suggesting to set Claw install directory.
  12. Reimplemented the automatic update functionality.
  13. Set "Add back and front planes" option in "New map" window pre-selected by default.

v0.3.2 (b31)

  1. Fixed welcome page buttons.
  2. Fixed the browse button in options, allowing to pick Claw install directory.
  3. Fixed crash when opening brush tool.
  4. Fixed crash when closing text easy editor without saving.
  5. Fixed crash when some settings are not set.
  6. Fixed crash when browsing empty tiles or images data bank.
  7. Fixed some other random bugs, which could impact the performance.
